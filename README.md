# 🌿 Paradise Nursery Shopping Application

Paradise Nursery is a dynamic e-commerce web application built using 
React and Redux Toolkit. The application allows users to browse a variety 
of houseplants, add them to a shopping cart, manage quantities, and view 
total pricing dynamically.

## 🚀 Features

- Browse plants grouped by categories
- View plant images, names, descriptions, and prices
- Add plants to cart
- Increase and decrease item quantity
- Remove items from cart
- Dynamic cart icon with item count
- Calculate total cart value
- Responsive navigation bar

## 🛠 Technologies Used

- React.js
- Redux Toolkit
- CSS
- JavaScript (ES6)

## 📦 Project Structure

- App.jsx – Landing page
- ProductList.jsx – Plant listing page
- CartItem.jsx – Shopping cart page
- CartSlice.jsx – Redux cart logic
- AboutUs.jsx – Company information
- App.css – Styling
